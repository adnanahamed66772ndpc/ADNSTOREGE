# D1 ডাটাবেস কিভাবে সেটআপ করবেন

D1 হলো Cloudflare-এর SQL ডাটাবেস। অ্যাপ বাকেট কনফিগ ও ফাইল মেটাডাটা এখানে জমা রাখে। একবার তৈরি করে প্রজেক্টের সাথে বাইন্ড করলেই হয়।

---

## ১. D1 ডাটাবেস তৈরি করা

### Cloudflare ড্যাশবোর্ড থেকে

1. [Cloudflare Dashboard](https://dash.cloudflare.com) এ গিয়ে লগইন করুন।
2. বাম পাশের মেনুতে **Workers & Pages** খুলুন (কোনো অ্যাকাউন্টে **Storage** → **D1** থাকতে পারে)।
3. **D1** ট্যাব বা **Create database** এ ক্লিক করুন।
4. **Create database** ক্লিক করুন।
5. একটা নাম দিন (যেমন `adnstorge-db`) এবং **Create** ক্লিক করুন।

### কমান্ড লাইন থেকে (Wrangler)

```bash
npx wrangler d1 create adnstorge-db
```

আউটপুট এরকম আসবে:

```
✅ Successfully created DB 'adnstorge-db'
database_id = "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx"
```

## ২. D1 ডাটাবেসের আইডি ও নাম কোথায় পাবেন

- **ডাটাবেসের নাম**: তৈরি করার সময় যে নাম দিয়েছিলেন (যেমন `adnstorge-db`)।
- **ডাটাবেস আইডি**: লম্বা আইডি, যেমন `xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx`।

**ড্যাশবোর্ডে কোথায় পাবেন:**

1. **Workers & Pages** → **D1** (বা **Storage** → **D1**) এ যান।
2. আপনার ডাটাবেসের নামে ক্লিক করুন।
3. **Database ID** ডাটাবেসের ওভারভিউ পেজে দেখাবে (কখনো "API" বা URL-এ থাকে)।
4. **নাম** উপরে টাইটেলে থাকে।

Pages-এ বাইন্ড করার সময় **নাম** দরকার। **আইডি** শুধু লোকাল ডেভের জন্য `wrangler.toml` এ দিলে লাগে।

---

## ৩. D1 অ্যাপের সাথে যুক্ত করা

### লোকাল ডেভেলপমেন্ট (wrangler.toml)

1. প্রজেক্টের রুটে **`wrangler.toml`** ফাইল খুলুন।
2. এই ব্লক খুঁজুন:
   ```toml
   [[d1_databases]]
   binding = "DB"
   database_name = "adnstorge-db"
   database_id = "placeholder"
   ```
3. **`database_name`** এ আপনার D1 ডাটাবেসের নাম দিন (যেমন `adnstorge-db`)।
4. **`database_id`** এ আপনার D1 **Database ID** দিন (`placeholder` এর জায়গায়)।
5. সেভ করুন। `npm run pages:dev` চালালে অ্যাপ এই ডাটাবেস ব্যবহার করবে।

### ডিপ্লয় করা সাইট (Cloudflare Pages)

1. ড্যাশবোর্ডে **Workers & Pages** → **Pages** → আপনার প্রজেক্ট এ যান।
2. **Settings** → **Functions** খুলুন।
3. **D1 database bindings** পর্যন্ত স্ক্রল করুন।
4. **Add binding** (বা থাকলে **Edit**) ক্লিক করুন।
5. সেট করুন:
   - **Variable name:** `DB` (ঠিক এই নামই হতে হবে)।
   - **D1 database:** লিস্ট থেকে আপনার ডাটাবেস সিলেক্ট করুন (যেমন `adnstorge-db`)।
6. **Save** ক্লিক করুন।
7. বাইন্ডিং কাজে লাগাতে প্রজেক্ট আবার ডিপ্লয় করুন।

---

## ৪. ওয়েব থেকে ডাটাবেস ইনিশিয়ালাইজ করা

D1 তৈরি ও বাইন্ড করার পর:

1. অ্যাপ খুলুন (যেমন http://localhost:8788 বা আপনার Pages এর URL)।
2. টেবিল না থাকলে আপনাকে **Database setup** এ নিয়ে যাবে।
3. **Initialize database** ক্লিক করুন।
4. অ্যাপ দরকারি টেবিল (buckets, files, settings) তৈরি করে দেবে। নিজে `wrangler d1 execute` চালানোর দরকার নেই।
